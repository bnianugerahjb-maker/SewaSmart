Bank Statement Import

Upload PDF/CSV, sistem extract column Date, Pay To, Description, Debit, Credit dan Balance.

Flow baru: Upload statement → Preview table keluar dalam pop-up → semak/edit column → pilih account code → post ke Accounting Ledger.
Nota: PDF bank statement yang berbentuk scan/gambar tidak boleh dibaca tanpa OCR. Untuk hasil terbaik guna PDF text-based atau CSV/export bank.

Auto Detect Column

Bahagian ini menggantikan matching lama yang susah faham.

Column yang dikesan:
Date, Pay To, Description, Debit, Credit dan Balance.

Money In / Credit:
Ledger post Dr Bank / Cr account code yang dipilih.

Money Out / Debit:
Ledger post Dr account code yang dipilih / Cr Bank.

User masih boleh edit nilai dalam preview sebelum post. Jika extract PDF tidak tepat, copy paste dari bank CSV/Excel lebih stabil.
